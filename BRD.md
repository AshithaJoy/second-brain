# Business Requirements Document (BRD)

# Second Brain

## Version 1.0 — Founder-Focused MVP Strategy

---

# 1. Summary

Second Brain is an AI-powered workflow assistant built for Instagram Reel creators.

The product helps creators go from:

* idea
  to:
* shoot-ready Reel plan

within minutes.

Second Brain simplifies the workflow between:

* trend discovery
* content ideation
* hook generation
* script creation
* B-roll planning
* shoot preparation

The first version of the product is intentionally narrow and focused.

Instead of building a large creator platform immediately, Second Brain focuses on solving one repeated creator workflow extremely well:

> helping creators consistently plan better Reels faster.

Second Brain is not designed to replace editing software like CapCut or Premiere Pro. Instead, it complements existing editing workflows by helping creators structure and prepare content before production.

The long-term vision is to gradually evolve into a broader creator workflow infrastructure platform after validating strong retention and workflow dependency.

---

# 2. Problem Statement

Instagram Reel creators currently use multiple disconnected tools and workflows (Notion, Google Docs, ChatGPT, CapCut, etc.) for:

* idea generation
* scripting
* trend analysis
* content structuring
* B-roll planning
* production planning

This creates:

* workflow friction
* inconsistent posting
* creator burnout
* poor execution speed
* weak planning systems
* scattered creator workflows

Most AI tools only generate generic text.

Very few tools help creators bridge the gap from:

> idea → structured shoot-ready workflow.

# 16. Risks & Mitigations

## Risk: High Creator Churn

### Mitigation

Focus on sticky production workflows like Shoot Mode and reusable workflow templates.

---

## Risk: High AI Costs

### Mitigation

Use credit-based usage limits, request caching, and lightweight template systems.

---

## Risk: Platform/API Restrictions

### Mitigation

Avoid dependence on direct scraping and use template-driven trend recreation.

---

## Risk: Overbuilding Before PMF

### Mitigation

Focus only on one repeated workflow until retention is validated.
